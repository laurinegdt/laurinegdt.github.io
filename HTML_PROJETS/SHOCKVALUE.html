PROJECT

SHOCK VALUE

skills : game development, motion capture, coding

tools :unreal engine, metahuman

When attention becomes currency, pain becomes entertainment.

An Unreal Engine project featuring a MetaHuman in a livestream. Likes from the audience unlock badges but also trigger escalating electric shocks. As engagement increases, so does the audience size, exposing a feedback loop where attention amplifies suffering. The project critiques the gamification of pain and questions the ethics of attention-driven digital entertainment.
