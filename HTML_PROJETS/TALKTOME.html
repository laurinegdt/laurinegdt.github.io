PROJECT

TALKTOME

skills : physical computing, 3D, coding, crafting

tools : IdeaMaker, VScode

A sugar daddy simulator.

An interactive object exploring dialogue as a playground, drawing inspiration from conversational interfaces to propose new and unconventional forms of interaction. This dialogue places the user in the role of a sugar daddy, tasked with balancing affection and financial resources within interactions.

Collaborative project with Media and Interaction designer : Dalia Spichtig
